## 三、宏观 Alpha 拆解

### 3.1 基础等式

```
净利润 = 理论利润 - 总费用 + Alpha
总利润 224,923 = 理论利润 - 340,493 + Alpha
理论利润 ≈ 565,416 U
```

### 3.2 Alpha 来源推测

从宏观层面，Alpha 不可能来自 L1 执行层（那层主要解决"上链"问题，不创造利润）。

真正的 Alpha 来源只能是：

| 来源 | 机制 | 说明 |
|------|------|------|
| **L3 卖币负滑点** | 触发条件满足 → 价格下跌 → 实际卖出价优于理论价 | 每日汇总中买币次数 ≈ 卖币次数，卖币的 +0.83 U / 笔抵消了买币的 -1.65 U / 笔 |
| **L2 低价差区间筛选** | 触发阈值 < 0.2% 时 slippage 接近 0 | 宏观上大部分订单在此区间 |
| **价格回归特性** | CEX-DDEX 价差会均值回归 | 这是策略存在的基础 |

## 五、结论

1. **策略盈利的根本原因**：捆绑费（占费用 94%）和 Gas 费是主要成本，但通过买卖方向不对称的滑点特性（卖币赚、买币亏），策略能以小额负 slippage 为代价换取整体正收益

2. **风险警示**：费用是利润的 1.5 倍，策略的 Alpha 缓冲空间有限，需严格控制 slippage 才能维持盈利

3. **优化方向**：降低捆绑费比例（如竞争者少时可调低）、严控价格阈值 < 0.2%、控制单笔规模 < 1k U
