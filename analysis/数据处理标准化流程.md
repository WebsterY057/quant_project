# 数据处理标准化流程

> 版本：v2.0 | 更新：2026-03-27

## 文件命名规范

### 合约类别
- `k1` — 跨1 合约
- `k2` — 跨2 合约

### 数据类别
| 类别 | 时间精度 | 日期是否重复 | 说明 |
|------|----------|-------------|------|
| **订单数据** | 毫秒 | 可重复 | 每笔交易明细 |
| **token数据** | 日 | 重复（每日多条） | 按 token 汇总的每日数据 |
| **日报数据** | 日 | 不重复 | 整体汇总，每日一条 |

### 命名格式
```
{合约}_{数据类别}_{日期}.xlsx
```

示例：
- `k1_订单数据_2026-03-24.xlsx`
- `k2_订单数据_2026-03-24.xlsx`
- `k1_token数据_2026-03-24.xlsx`
- `k1_日报数据_2026-03-24.xlsx`

### 原始导出命名
```
k1_export_orders_YYYYMMDD.xlsx   ← 订单数据原始导出
k1_token_YYYYMMDD.xlsx           ← token数据原始导出（待确认）
k1_daily_YYYYMMDD.xlsx           ← 日报数据原始导出（待确认）
```

## Python 处理代码模板

```python
import pandas as pd

# ========== 配置 ==========
# 命名格式: {合约}_{数据类别}_{日期}.xlsx
# 合约: k1 / k2
# 数据类别: 订单数据 / token数据 / 日报数据

CONTRACT = 'k1'           # k1 或 k2
DATA_TYPE = '订单数据'      # 订单数据 / token数据 / 日报数据
DATE = '2026-03-24'

RAW_FILE = f'{CONTRACT}_export_orders_20260324.xlsx'       # 原始导出
OUTPUT_FILE = f'{CONTRACT}_{DATA_TYPE}_{DATE}.xlsx'        # 清洗后输出

# ========== Step 1: 读取 + 列名映射 ==========
COLUMN_MAP = {
    'time': '时间',
    'block': '区块号',
    'path': '交易对',
    # ... 完整映射见上方 Step 1 表格
}

df = pd.read_excel(RAW_FILE)
df = df.rename(columns=COLUMN_MAP)

# ========== Step 2: 删除多余列 ==========
DROP_COLS = [
    'id', 'token_decimals', 'stable_decimals', 'hash_num',
    'w1_uw_maxpair', 'w1_reserve', 'w1_uw_reserve',
    'w2_uw_maxpair', 'w2_reserve', 'w2_uw_reserve',
    'bundle_rate', 'profit_virtual_fee'
]
df = df.drop(columns=[c for c in DROP_COLS if c in df.columns])

# ========== Step 3: 数据类型校正 ==========
df['时间'] = pd.to_datetime(df['时间'])
df['区块号'] = df['区块号'].astype(int)
df['状态'] = df['状态'].astype(int)

# ========== Step 4: 异常值处理 ==========
# 过滤成功订单用于主分析
df_success = df[df['状态'] == 1].copy()

# ========== Step 5: 保存 ==========
df_success.to_excel(OUTPUT_FILE, index=False)
print(f"清洗完成: {len(df_success)} 条成功订单 → {OUTPUT_FILE}")
```
